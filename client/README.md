# ✨ Eras Music Practice Suite ✨
A Full-Stack (MERN) application for musicians to track practice sessions with a Taylor Swift inspired aesthetic.

## 🚀 Features
* **Custom Audio Engine:** Built with Web Audio API for Tuners and Tone Generators.
* **RESTful API:** Node.js/Express backend for managing practice logs.
* **Secure Auth:** User authentication using **JWT**.
* **Database:** Persistent storage with **MongoDB Atlas**.
* **Theming:** Dynamic "Eras" theme engine (Midnights vs. Reputation).

## 🛠️ Tech Stack
* **Frontend:** React.js, Vite, CSS3 (Glassmorphism)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose ODM)
* **Auth:** JSON Web Tokens (JWT), Bcrypt
